# Bundled fonts

Slumbr bundles two open-source typefaces, both under the **SIL Open Font
License 1.1**. The full license for each ships alongside it here, as the OFL
requires when redistributing the font files.

| Font | Role | Copyright | License |
| --- | --- | --- | --- |
| **Inter** (`Inter-Regular.ttf`, `Inter-SemiBold.ttf`) | UI / body | © 2016 The Inter Project Authors — https://github.com/rsms/inter | [OFL-1.1](OFL-Inter.txt) |
| **Sora** (`Sora.ttf`) | Display / headings | © 2019 The Sora Project Authors — https://github.com/sora-xor/sora-font | [OFL-1.1](OFL-Sora.txt) |

The OFL permits bundling and redistribution; it forbids selling the fonts on
their own and reserves the reserved font names. Slumbr ships them only as part
of the application UI.
